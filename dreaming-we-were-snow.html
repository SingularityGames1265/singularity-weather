The Babel Effect

The Void Plane features many aspects from earth. This includes text as well. But any text, be it on a sign, product or other place containing written text is jumbled. Where letters could be expected, a random symbol is placed. This can include any letter from the Latin alphabet, puncuation marks or spaces. If a logo found in the Void Plane would contain text, those letters would be changed as well.

The effect got its name from the philosophical thought experiment called "The Library of Babel". In this library, books of 410 pages reside. Each page holds 40 lines of 80 random characters. All possible variations exist within this library. This means that everything that was, is and will be written can be found here. But many of these books are unreadable, having only a few legible words written in them.
